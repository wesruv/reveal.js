The Hardware

Low-End Headsets

  • Sitting/Standing only
  • $20 to $100
  • Limited by Phone Hardware

Higher End Headsets

  • Room Scale and Standing/Sitting
  • VR Controllers
  • $500 to $800
  • Requires powerful computer

Applications for VR

  • Gaming, Photos, Video
  • Demonstrate macro/micro scale
  • Tour Museums, Distant Locations, Event Re-enactment
  • Therapy for PTSD, Phobias and Brain Injuries
  • Training simulation

Web VR

Moz VR

  • Goal: bring VR to web browsers natively
  • Working on a WebVR API W3C specification
  • Working on frameworks/libraries that can be used today

WebVR Boilerplate

  • Starter kit for custom WebVR/3D experience in modern browsers
  • Very JS Heavy
  • Developer centric

A-Frame

  • Built on WebVR Boilerplate
  • Easy to learn markup language
  • Can further be modified with JS
  • Project goal is to enable more creators
